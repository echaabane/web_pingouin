Secret Key et autres variables à cacher

Il faut cacher des clés, des passwords, des users. Mais je veux mettre mon code sur Git Alors comment faire?
Créez un fichier .env et y mettre en clair vos clés. Exemple : SECRET_KEY = 1234
Conseil: mettre une meilleure clé que 1234
Ensuite avec la bibliothèque OS vous pouvez demander une clé dans la variable d'environnement du server.

from dotenv import load_dotenv
import os

load_dotenv()

#recup clé secrete dans un fichier .env
SECRET_KEY = os.getenv("SECRET_KEY")
Il ne vous reste plus qu'à ajouter une variable d'environnemt dans votre serveur.
Sur Heroku vous le trouverez dans setting reveal config vars
