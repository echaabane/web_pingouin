Aide Mémoire AWS_CDK

Aws cdk permet de faire de l'Infrastructure As A Code.
Grâce à du code python, vous pouvez créer des infrastructure dans aws.

Installation et configuration aws_cdk
prérequie: Il faut créer un compte et un rôle IAM qui permettent de créer les instances sur aws.
Il faut aussi installer aws CLI (aws en ligne de commande).
Première étape : se créer un dossier pour stocker IAAC

cdk init app -- language python
Permet d'initialiser un répertoire. (Effectue aussi un dépôt git local)
Créer un environnement virtuel
python -m venv venv
source /venv/bin/activate
pip install -r requirements.txt
Si ce n'est pas déjà fait : configurer son compte aws pour CLI :
aws configure ()
Pour le reste de ce tuto on va directement sur le git de AWS
