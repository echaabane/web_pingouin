Aide mémoire git

Rappel des commandes de base de Git
Initialisation du répertoire local :

git init -b main

init : crée un repertoire .git avec le depôt local de git
-b nom_de_branche : permet de nommer le nom de la branche principale (par défaut master)


ajout de fichier à tracker :
git add nom_fichier


Ignorer des fichiers ou des dossiers :
Créer un fichier .gitignore et mettre les fichier à ignorer :
exemple :
.env
/dossier


Commiter le code :
git commit -am "commentaire du commit"


Créer une branche :
git branch nom_de_la_branche


Aller dans une branche :
git checkou nom_branche


Effacer une branche :
got branch -D nom_branche
Merger une branche :
Attention se placer dans la branche à merger
merge de dev vers main : se placer dans main
git merge nom_de_la_branche


Mettre à jour une branche par rapport à une autre.
REBASE main vers dev
git rebase main dev


Cloner un repo :
git clone path_du_repository_ou_du_gist path_du_repertoire_a_creer_en_local


Lier git et github :
git remote add origin https://github.com/nomutilisateur/MonProjet.git


Push vers repo :
git push origin nom_branche_a_pusher
