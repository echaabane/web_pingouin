Introduction aux bibliothéques python pour la datascience

Description des bibliothéques de base pour la data science :
Ceci est juste une introduction pour les bibliothéques. Vous pouvez cliquer sur le lien vers la doc en début de chapitre.

Les bibliothéques pandas et numpy :



Numpy

Numpy est une bibliothèque orientée math. L'objet le plus utile est l'array ou le tableau numpy.
Un array est une matrice : une sorte de tableau rempli de chiffre.
Ces tableaux vous permettent de faire du calcul matriciel.
Un petit rappel math : cours matrice.
Interêt : format léger en mémoire et rapide. Utile en Machine Learning.

Pandas


Pandas est une bibliothèque utilisant des dataframes.
Un dataframe est une sorte de tableau excel ou de table de base de données.
Vous pouvez créer un dataframe avec des fichiers (excel,csv,...) ou en vous connectant avec certaines BDD.


Les bibliothèques d'affichage :


Matplotlib



Matplotlib est une bibliothèque qui permet entre autre de créer des graphes.
Il y a 2 façons de l'utliser :
La méthode script
La méthode objet
Les 2 méthodes fonctionnent, mais j'ai une préfèrence pour la programation orientée objet.

Seaborn



Seaborn permet de faire des graphes. Cette bibliothèque est liée à Matplotlib. Vous pouvez passer un objet axe maplotlib dans les fonctions Seaborn. Seaborn prend mieux en charge les dataframes que Matplotlib.

Les statistiques :



Statsmodel


Statsmodels permet de faire des tests statistiques, des régressions. Bref c'est une bibliothèque pour statisticiens.

Scikit-learn



ou Sklearn permet, elle aussi, de faire des tests statistiques.

Les bibliothèques IA :

Scikit-learn



Ou Sklearn permet de bien débuter dans les IA.
Vous avez accès à des outils pour normaliser les données, des petits models d'IA.

Tensorflow



Tensorflow permets de créer des algorythmes de ML. Il peux utiliser des CPU ou des GPU.
